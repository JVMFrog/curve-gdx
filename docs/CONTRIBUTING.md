# Contributing

## Clone repository

```shell
mkdir curve-lib-src
cd curve-lib-src
git clone https://github.com/JVMFrog/curve-gdx
git clone https://github.com/JVMFrog/curve-gdx-examples
```

Then open `curve-gdx-examples` in IDEA

This will load the library and examples module as one project:
